task
1. Создайте новый репозиторий на GitHub и назовите его "как угодно"
2. В этом репозитории создайте файл "hello.txt"
3. Внесите в файл "hello.txt" следующий текст  "что угодно" 
4. Создайте копию (ветку) репозитория
5. Внесите изменения в файл "hello.txt"  например, замените "World" на своё имя
6. Сохраните изменения
7. Объедините изменения из копии с оригинальным репозиторием

solve
1. Перейдите на GitHub и создайте новый репозиторий с именем "как угодно"
2. Внутри репозитория нажмите на "Add file" и выберите "Create new file". Назовите файл "hello.txt" и напишите в нем "что угодно"
3. Создайте копию репозитория, нажав на кнопку ""
4. Внутри копии репозитория найдите файл "hello.txt" и отредактируйте его, заменив "World" на своё имя. Сохраните изменения
5. Обратитесь к оригинальному репозиторию и нажмите "New Pull Request", чтобы объединить изменения из копии
6. Подтвердите запрос на объединение изменений
